## 项目解决的问题
想要解决普通的个人代码在多台电脑的倚赖问题
#### 实现的方法
设立一个服务中心，接受任意的上传下载请求，并且通过json文件来配置版本信息
- 上传
上传的时候，先通过json文件与服务器交流,需要的文件
- 下载
